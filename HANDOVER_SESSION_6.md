## ✅ ЧТО СДЕЛАНО В СЕССИИ 5

### 1. Архитектура меню (ПРИОРИТЕТ 1)

- Режим модерации только для `pm` и `admin` через Настройки

### 2. Разделение пользовательского и режима редактора (ПРИОРИТЕТ 2)

### 3. Роль-зависимые чеклисты (ПРИОРИТЕТ 3)
- Каждая роль видит только свои чеклисты
- Структура: barman → opening+closing+bar, waiter → opening+closing+floor и т.д.
- **Файлы:** `bot/handlers/tasks.py`, `bot/handlers/library.py`

- **Файлы:** `menu_loader.py`, `menu_database.sql`, `menu_report.md`
- **Таблицы:** categories, dishes, drinks, allergens, dish_allergens

### 5. Регламенты (перенос стандартов)
- Стандарты и принципы работы бара → Регламенты

- Новые категории в регламентах: "🍸 Стандарты бара", "🏆 Стандарты RAMO"
- **Файлы:** `bot/utils/cache_manager.py`

### 6. Праздники (объединение)
- Объединены "Праздники месяца" и "Дни рождения" в один раздел

### 7. Управление задачами (расширение)

- 📊 Журнал задач с фильтрацией:

## 🏗 АРХИТЕКТУРА БОТА (актуально на 23.07)

### Структура папок
```
bot/
├── handlers/
│   ├── registration.py     ← авто-одобрение при регистрации
│   ├── menu.py             ← главное меню + Настройки
│   ├── library.py          ← Библиотека (чеклисты, роли, регламенты, меню кухни/бара)
│   ├── promos.py           ← Акции
│   ├── task_manager.py     ← Управление задачами (создание, выполнение, журнал)
│   ├── tasks.py            ← Чеклисты (интерактивные)
│   ├── handover.py         ← Передача смены
│   ├── incident.py         ← Инциденты
│   ├── events.py           ← События (брони, афиша, праздники)
│   ├── admin.py            ← Панель управления (режим модерации)
│   └── control.py          ← Контроль команды
├── utils/
│   ├── cache_manager.py    ← _SEED с контентом (роли, регламенты, чеклисты)
│   ├── db_connector.py     ← БД: User, ChecklistExecution, Task, Event и т.д.
│   └── promo_scheduler.py  ← фоновая рассылка акций
├── states/forms.py         ← FSM состояния
├── keyboards/inline_buttons.py ← кнопки
└── main.py                 ← точка входа
```

### Роли и доступ
| Роль | Режим модерации | Видит чеклисты | Может ставить задачи | Может выполнять задачи |
|------|-----------------|----------------|----------------------|----------------------|
| pm   | ✅ Да          | ✅ Все        | ✅ Да                | ✅ Да (свои)        |
| admin| ✅ Да          | ✅ Все        | ✅ Да                | ✅ Да (свои)        |
| manager| ❌ Нет       | ✅ Все        | ✅ Да                | ❌ Нет              |
| barman| ❌ Нет        | По роли       | ❌ Нет               | ✅ Да               |
| waiter| ❌ Нет        | По роли       | ❌ Нет               | ✅ Да               |
| security| ❌ Нет      | По роли       | ❌ Нет               | ✅ Да               |
| cleaning| ❌ Нет      | По роли       | ❌ Нет               | ✅ Да               |
| cook/chef| ❌ Нет     | По роли       | ❌ Нет               | ✅ Да               |

### Callback префиксы
- `menu:` — главное меню
- `lib:` — Библиотека
- `tm:` — управление задачами (менеджер)
- `tme:` — редактирование задач
- `mt:` — мои задачи (работник)
- `mtc:` — выполнение задачи (работник)
- `admin:` — панель администратора
- `events:` — события
- И т.д.

## 🔧 ДАННЫЕ БД

### Menu.db (55 позиций)
- `categories` — категории (Супы, Кофе и т.д.)
- `dishes` — 28 блюд кухни (price, is_spicy, is_vegetarian, is_new, is_healthy)
- `drinks` — 27 напитков бара (price, volume_ml, is_non_alcoholic, и т.д.)
- `allergens` — 8 аллергенов (орехи, глютен, лактоза и т.д.)
- `dish_allergens` — связи между блюдами и аллергенами

**Теги:**
- Острое (🌶️) → is_spicy
- Вегетарианское (🥗) → is_vegetarian
- Новинка (✨) → is_new
- ЗОЖ (💪) → is_healthy
- Безалкогольное (☕) → is_non_alcoholic (для бара)
